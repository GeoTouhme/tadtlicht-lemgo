# Stadtlicht Lemgo

A modern, responsive web application built for Stadtlicht Lemgo. This project showcases a premium user interface with dynamic animations and a focus on visual excellence.

## Project Vision

### The Challenge
In an era where digital first impressions determine business success, many local establishments rely on outdated, static websites that fail to capture their unique atmosphere. Standard template branding often results in generic user experiences, poor performance on mobile devices, and a disconnect between the physical and digital brand identity.

### The Solution
Stadtlicht Lemgo solves this by delivering a **bespoke, high-performance web experience** acting as a direct digital extension of the venue. Unlike generic website builders, this application offers:
-   **Immersive Storytelling:** Through video backgrounds and motion-enhanced interactions.
-   **Instant Performance:** Leveraging Next.js server-side rendering for immediate page loads.
-   **Premium Availability:** A fully accessible, responsive interface that works flawlessly across all devices.

### Why Development?
Developing this platform is crucial to set a new standard for local business digitalization. It serves not just as an informational hub, but as a **customer acquisition tool** that conveys quality and professionality before a guest even steps through the door.

## Technologies

This project is built using the latest web technologies to ensure performance, scalability, and a great developer experience:

-   **[Next.js 16](https://nextjs.org/)** - The React Framework for the Web (App Router).
-   **[React 19](https://react.dev/)** - The library for web and native user interfaces.
-   **[TypeScript](https://www.typescriptlang.org/)** - For type safety and better developer tooling.
-   **[Tailwind CSS v4](https://tailwindcss.com/)** - A utility-first CSS framework for rapid UI development.
-   **[Framer Motion](https://www.framer.com/motion/)** - A production-ready motion library for React.

## Features

-   **Modern Design**: A clean, aesthetic interface utilizing modern design principles.
-   **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices.
-   **Dynamic Animations**: Includes custom animations like the Typewriter effect for engaging user interaction.
-   **Video Backgrounds**: High-quality video integration for immersive hero sections.
-   **Component-Based Architecture**: Modular and reusable components (Navbar, Hero, Footer, etc.).

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/GeoTouhme/tadtlicht-lemgo.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd tadtlicht-lemgo
    ```
3.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

### Running the Development Server

Start the local development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `app/`: Contains the main application routes and pages (App Router).
-   `components/`: Reusable UI components (Navbar, Hero, etc.).
-   `public/`: Static assets like images, videos, and fonts.
-   `styles/`: Global styles and Tailwind configuration.
